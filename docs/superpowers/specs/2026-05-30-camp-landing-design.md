# Спецификация: Лендинг церковного детского лагеря

**Дата:** 2026-05-30
**Статус:** Утверждён к реализации

## 1. Цель и контекст

Ежегодный церковный детский лагерь **с проживанием** для детей **8–13 лет**. Лагерь
проходит в **Храздане**, длится **5 дней**. Участвуют дети из **Аштарака** и сёл
**Кош** и **Оганаван**.

**Проблема:** каждый год не удаётся набрать достаточно детей.

**Решение:** одностраничный сайт (лендинг), который не просто *рассказывает*, а
**показывает** лагерь живыми фото и видео, и позволяет родителям заранее подать
заявку через форму регистрации.

**Аудитория:** в первую очередь **родители** (принимают решение, тревожатся за
безопасность ребёнка) и сами дети (нужно «зажечь» интересом). Сайт должен
одновременно вызывать доверие у родителей и радость у детей.

**Язык сайта:** армянский. (Эта спецификация написана по-русски для заказчика;
весь текст на сайте — на армянском.)

## 2. Технический подход

- **Тип:** один статический HTML-файл + CSS + минимум JS. Без фреймворков, без
  шага сборки — чтобы легко поддерживать и обновлять каждый год.
- **Mobile-first, адаптивный.** Приоритет — идеальное отображение на телефоне
  (большинство родителей откроют с телефона), затем десктоп.
- **Производительность:** быстро грузится даже на слабом интернете
  (оптимизированные фото, lazy-loading галереи и видео).
- **Хостинг:** бесплатный — **Netlify** или **GitHub Pages**. Бесплатный адрес
  вида `имя.netlify.app`. Свой домен можно подключить позже.
- **Переиспользуемость:** контент (даты, фото, тексты) вынесен так, чтобы каждый
  год можно было легко обновить без переписывания кода. Места для правки будут
  подписаны комментариями.

## 3. Визуальный стиль

Утверждённое направление — «**спокойнее**» (вариант B из брейнсторма):

- Светлый тёплый фон (кремово-белый), много «воздуха».
- Закатный акцент — мягкий оранжево-золотой градиент в шапке.
- Мягкие округлые формы (дружелюбно, но не «детсадовски»).
- Один основной акцентный цвет, минимум эмодзи.
- **Живые фото — главный элемент.** Дизайн намеренно сдержанный, чтобы фото
  «играли» сильнее, а у родителей возникало доверие.

Ориентировочная палитра: фон `#fdf8f0`, акцент-градиент `#e9742f → #f0a35a`,
тёмный текст `#6b4a2e`.

## 4. Структура страницы (блоки сверху вниз)

1. **Главный экран (Hero)** — большое фото/видео счастливых детей. Заголовок,
   подзаголовок. Бейдж «Бесплатно — платится только проезд». Краткие факты:
   возраст 8–13, с проживанием, 5 дней, Храздан. Кнопка «Գրանցվել» (Регистрация).
2. **Что за лагерь** — тёплый короткий текст + карточки ценностей
   (вера, игры/природа, новые друзья, безопасность).
3. **Условие участия** — крупно и понятно: «Лагерь для детей из Аштарака, Коша
   и Оганавана».
4. **Галерея фото** — сетка живых фото с прошлых лагерей. Сердце страницы.
5. **Видео** — ролик с прошлого лагеря (атмосфера за ~1 минуту).
6. **Программа на 5 дней** — как проходят дни: утро, занятия, игры, вечер.
7. **Проживание и безопасность** — где спят, питание, кто присматривает за
   детьми. Снимает тревогу родителей.
8. **Наши вожатые (руководители)** — **без фото и без имён**. Текстовый блок о
   том, что у руководителей многолетний опыт работы с детьми, они ответственные
   и проверенные. Цель — доверие родителей.
9. **Что взять с собой** — список вещей, которые ребёнок берёт в лагерь.
10. **Стоимость** — крупно: лагерь бесплатный, оплачивается только проезд
    **3000–4000 драм**.
11. **Отзывы** — блок **заложен в вёрстку, но скрыт/пустой**. Заказчик заполнит
    позже (текст отзыва, имя). Должен легко включаться без переделки.
12. **FAQ** — частые вопросы родителей (связь с ребёнком, безопасность, кто
    проводит и т.п.).
13. **Форма регистрации** — см. раздел 5.
14. **Контакты / футер** — адрес, карта, телефон, ссылки на соцсети церкви.

**Дополнительно:** на телефоне — «липкая» кнопка «Գրանցվել» внизу экрана, чтобы
регистрация всегда была под рукой.

## 5. Форма регистрации

**Назначение данных:** все заявки попадают в **Google-таблицу**.

**Поля:**
- Имя и фамилия ребёнка — *обязательно*
- Возраст — *обязательно*
- Имя родителя — *обязательно*
- Телефон родителя — *обязательно*
- Откуда — *обязательно*, **выбор из списка**: Аштарак / Кош / Оганаван
- Здоровье / аллергии — необязательно (важно для лагеря с проживанием: питание,
  лекарства)
- Копия свидетельства о рождении — **необязательно**, загрузка файла. Подпись:
  «можно загрузить сейчас или принести копию при заезде».
- Комментарий — необязательно
- Галочка согласия родителя — *обязательно*

**Поле «церковь/община» — НЕ включаем** (по решению заказчика).

**Технология формы:**
- Форма отправляет данные в **Google Apps Script (Web App)**, который дописывает
  строку в Google-таблицу. Бесплатно, без сторонних сервисов.
- **Загрузка файла:** файл свидетельства сохраняется в **приватную папку Google
  Drive**; в таблице записывается ссылка на файл. Доступ к папке — только у
  заказчика.
- После отправки — понятное сообщение об успехе на армянском.
- Базовая валидация полей на стороне клиента (обязательные поля, формат телефона).

## 6. Проверка «свой / чужой» (проблема с местом проживания)

Сайт технически не может проверить, действительно ли семья из Аштарака/Коша/
Оганавана. Защита — организационная, в три слоя:

1. **Явное условие** на сайте (блок 3) — большинство «чужих» не подают.
2. **Поле «Откуда» — список из 3 мест**, а не свободный текст.
3. **Человеческое подтверждение:** форма = *предварительная заявка*. Координатор
   перезванивает родителю и подтверждает участие; церковная сеть/пасторы помогают
   перепроверить семьи. Финальное зачисление — за людьми, не за сайтом.

## 7. Приватность и данные

- Свидетельство о рождении и контакты — персональные данные. Хранить в закрытой
  Google-таблице и приватной папке Drive с ограниченным доступом.
- Галочка согласия родителя обязательна перед отправкой.

## 8. Контент, который заказчик предоставит (в вёрстке — заглушки)

- Фото и видео с прошлых лагерей (есть, хорошего качества).
- Точные даты лагеря (5 дней).
- Тексты блоков на армянском (или заказчик утверждает черновики).
- Список вещей «что взять с собой».
- Телефон и контакты, ссылки на соцсети, адрес для карты.
- Реальные отзывы (позже, в скрытый блок).

## 9. Вне рамок (YAGNI)

- Нет личного кабинета, авторизации, оплаты онлайн.
- Нет CMS/админки — обновление контента правкой файла (с понятными метками).
- Нет многостраничности — всё на одной странице.
- Нет автоматической проверки места жительства (делается людьми).
- Многоязычность не требуется (только армянский).
